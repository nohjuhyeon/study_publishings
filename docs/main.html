HTML 실습

구분 내용 링크
h1 제목과 내용 입력을 배움 temp로 이동
div 줄바꿈 기능을 배움 welcome로 이동
table 테이블 작성 방법을 배움 tables로 이동
a:link 링크 설정 방법을 배움 link로 이동
img 이미지 넣는 방법을 배움 Images로 이동
li 리스트 작성 방법을 배움 lists로 이동
form_input input 사용법을 배움 forms_inputs로 이동
form_input_type 다양한 input 사용 방법을 배움 form_input_types로 이동
select select 사용 방법을 배움 forms_selects로 이동

CSS 실습

구분 내용 링크

border 태그 자체에 색깔 적용 방법을 배움 borders로 이동
